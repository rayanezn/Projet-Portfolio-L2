# Portfolio de Rayane Zane

Ce portfolio a été réalisé dans le cadre de ma Licence 2 en informatique.  
Il présente mes projets, mes compétences, ainsi que mes informations de contact.

## Technologies utilisées
- HTML, CSS, JavaScript
- Bootstrap pour la mise en page responsive
- Animate.css pour les animations
- FontAwesome pour les icônes sociales

> Note : Ce portfolio a été créé lors de ma L2, il n’est donc pas forcément à jour avec mes compétences actuelles.

## Contenu du portfolio
- **Accueil** : Présentation personnelle avec photo, description et lien pour télécharger le CV  
- **Projets** : Galerie de projets avec descriptions et liens GitHub  
- **Compétences** : Liste de mes compétences techniques et outils maîtrisés  
- **À propos** : Informations sur mon parcours académique et mes objectifs  
- **Contact** : Formulaire pour me joindre directement

## Lancer le portfolio
Pour visualiser le portfolio, il suffit d’ouvrir le fichier `index.html` dans un navigateur web.
